# Interactive 3D Knowledge Graph Builder (PS9)

This project automatically builds an interactive 3D knowledge graph of a student's
skills and learning progression by aggregating data from GitHub and other sources.

## Modules
- ML Engine: Skill extraction & progression (NLP)
- Backend: APIs, auth, data orchestration
- Graph Engine: Knowledge graph construction
- Frontend: Interactive 3D visualization
